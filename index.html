近期个人作品

ryan-cli

这是一个nodejs制作的命令行工具，主要是为了解决日常工作中，提高开发功能重复的工作，可以根据这个脚手架创建页面模板，目前集成了vue的脚手架vue-cli,和react脚手架create-react-app,提高开发效率，目前托管在npm上，可以通过npm install -g ryan-cli测试功能

另外一个功能就是一个mock-server 通过ryan watch db.json的命令开启一个restful mock server

实现原理:是通过express 对一个json文件进行读写操作，返回特定字段的数据，也可以提供随机数据

项目目的:解决工作中需要mockjs的配置繁琐

项目经验

安全中心流量项目

MIUI安全中心的流量校正功能有一个购买流量的入口，我们与移动，电信，联通三大运营商合作售卖前向流量，后向流量与定向流量

这是个历史项目，我的角色属于维护工作，主要技术是jquery与grunt

小米手机安全中心游戏加速项目

小米手机安全中心有较高的系统权限通过提高用户手机的CPU使用率和杀死其它进程，来提高手机硬件性能，为用户提供游戏加速功能

这是2016年中的一个项目，我的角色是，中途接手这个项目，主要技术是VUE

安全组CDN与前端持续发布项目

我为MIUI做了个类似windows操作系统的用户体验的文件系统，用户可以上传文件与文件夹到我为这个用户所分配的目录，文件存在小米云的某个某个集群服务器。并且根据这个CND项目后期开发了持续发布项目 主要功能是用户通过创建个人项目，主要是根据用户传入的gitlab地址，在远程机器进行shell的git pull操作，并且讲git pull下来的项目文件与数据库关联。用户可以根据前端的pull按钮，切换分支按钮，部署前端项目 这个项目的主要技术用了，react redux django mysql nodejs全部由我完成。
